2) Alter : Alter command is most important part of DDL commands 

    using of Alter command we can add , remove or modify fields of table 

    1) add     - to add specific new field in existing table 
    2) drop   - to remove specific column 
    3) modify  - to modify filed datatyppe or size 
    4) rename   - to rename fieldname 

    e.g. 
    syntax : 

        alter table <tablename>  <altercommand>   fieldname  datatype 

        e.g. 

        -> alter table  student  add  city  varchar(20); 

        -> desc student;

        -> alter table  student  <drop>  fieldname; 

        e.g.  alter table student add dummy varchar(20);

        -> desc student;
        ->  alter table student drop dummy; 
        -> desc student;

        modify : when we want to change any field data type or field size we can 
            use alter modify 

        syntax : 

            alter table <tablename> modify <columnname> <datatype> ;

            e.g. 

            alter table student modify name varchar(25);

            -> show databases;
            -> use 8decse;
            -> show tables;
            -> desc student;
            -> alter table student modify city varchar(30);
            -> desc student;


    => alter rename command 

            when we want to change column name using of rename command 

            syntax : 
                alter table <tablename> rename column  <columname> to <new columnname>; 

            e.g. 

            ->  alter table student rename column score to marks;
            -> desc student;

4) Truncate  :

        using of Truncate command we can remove all records from the table

        drop command we have to create table first then re enter data. 

        Truncate is faster then drop because it delete all records from the table. 

        e.g.  drop : account delete 

              Truncate : delete order history or search history ...

            # Truncate table <tablename>;

1) NOT NULL : when we add any field as a not null means at a insert time we can't skip this field. 

        bydefault all fields are null if we want to specify specific field 
        with not null we can follow below syntax : 

        syntax : 

            create table <tablename>
            {
                fieldname datatype(size)  not null,
                .
                .
            };

                id int,

        using of primary key we can make any column as unique column but 

        primary key we can apply only on single field of table. 

            unique                      primary key 

        many column                     only one column 
        can accept blank / null         not accept null/blank value 

        e.g. email,phone,               e.g.  id , rollno, enroll_id..etc 
             adharcard id ,uuid

            create table <tablename>

                fieldname  datatype  primary key,
                .

iii)  DQL : Data Query Language : 

            most most powerfull concept of sql 

            when we want to retrive records from the table we can use DQL command \

                -> select 

    using of select command we can retrive different different records with specific condition wise. 

            syntax : 

            -> select * from <tablename>;

            e.g. -> select * from student;

    fetch specific columns from table 

            syntax : 

            -> select fieldname1,fieldname2... from <tablename>;

            e.g.  select name,subject from student;

    fetch specific data using of specific condition wise 
            for condition we use where clause. 

            syntax : select * from <tablename> where condition.;

            e.g. select * from student where city="ahmedabad";

            -> select name,subject,city from student where city="ahmedabad";

    conditional operator : 

            = , != , > , < , >= , <= 

    logical operator : 
            and , or , not 
        
    membership operator : 
            in , not in , between
        
    pattern matching operator : 
            like , not like 
        
    null operator : 
            is null , is not null

Order by : when we want to retrive data from the database with specific orderformate 

        by default order format is ASCENDING order 

        there are 2 keywords  ASC and DESC 

        syntax : 

        select * from <tablename> order by <columname>;
        or 
        select * from <tablename> order by <columname> asc;
        or 
        select * from <tablename> order by <columname> desc;

    e.g.  -> select * from student order by name;
          ->  select * from student order by marks asc;
          -> select * from student order by marks desc;
          ->  select * from student order by marks desc limit 3;
          ->  select * from student order by name,subject;

GROUP BY : group by row wise which contain common column value 

    to use concept of GROUP BY we have to use aggregate functions 

    -> COUNT(*) - count no. of record 
    -> SUM : total 
    -> MAX : maximum value
